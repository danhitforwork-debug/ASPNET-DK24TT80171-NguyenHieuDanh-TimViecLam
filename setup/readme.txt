HUONG DAN CHAY WEBSITE TIM VIEC LAM

1. Mo SQL Server Management Studio.

2. Mo file Database.sql trong thu muc setup.

3. Bam Execute de tao database JobRecruitmentDb.

4. Mo Visual Studio.

5. Chon File > Open > Web Site.

6. Chon thu muc:
   scr/JobRecruitmentWebSite

7. Mo file Web.config va kiem tra connectionString.

Neu dung SQL Express:
Data Source=.\SQLEXPRESS

Neu dung LocalDB:
Data Source=(localdb)\MSSQLLocalDB

8. Chay project bang IIS Express.
