HUONG DAN CHAY WEBSITE TUYEN DUNG VIEC LAM - BAN DE MO

1) MO SOURCE
- Mo Visual Studio 2022
- Chon File > Open > Web Site...
- Chon thu muc JobRecruitmentWebSite_Easy
- KHONG mo file .sln vi ban nay khong can .sln

2) TAO DATABASE
- Mo SQL Server Management Studio (SSMS)
- Connect vao .\SQLEXPRESS hoac (localdb)\MSSQLLocalDB
- Mo file Database.sql
- Bam Execute

3) SUA CHUOI KET NOI NEU CAN
Mo Web.config, neu dung LocalDB thi doi thanh:
Data Source=(localdb)\MSSQLLocalDB;Initial Catalog=JobRecruitmentDb;Integrated Security=True

4) CHAY WEB
- Trong Visual Studio bam chuot phai Default.aspx
- Chon Set As Start Page
- Bam F5
